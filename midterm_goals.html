    طراحی و راه اندازی واحد پردازش فوق سریع(اَبَر‌رايانش) جهت مقاصد تحقيقاتي

    سیستم بر مبنای Scyld ClusterWare™ HPC طراحی و پیاده سازی خواهد شد.
    جهت شروع کار ابتدا 32 کامپیوتر پنتیوم با سرعت 2.4Ghz با مقدار 512مگا بایت حافظه اصلی بعنوان گره های محاسباتی که با یک واسط شبکه 100مگابیتی به یک سوئیچ لایه 3 سیسکو مدل 3560 با قابلیت بک پلن 32mpls متصل میگردند و از طریق گره اصلی (master node) از طریق پروتکل BOOTP بوت میشوند و نیازی به نصب سیستم عامل بروی گره های محاسباتی نمیباشد.و این پروسه بگونه ای میباشد که هیچ گونه نرم افزار و راه انداز اضافی را که سبب صرف منابع میگردد ، شامل نشود. و در نهایت کل سیستم از دیدگاه کاربر به عنوان یک سیستم واحد دیده شود.
    ساختار نرم افزاری:

     Scyld as ClusterWare
     Red Hat Enterprise Linux ES 4. as OS
     web-based monitoring tool Ganglia
     Optimized open source MPI libraries
     freeNAS as Storage system
     bproc - the Beowulf process migration technology; an integral part of Scyld Beowulf
     beosetup - a GUI interface for configuring the cluster
     beostatus - a GUI interface for monitoring cluster status
     beostat - a text-based tool for monitoring cluster status
     beoboot - a set of utilities for booting the compute nodes
     beofdisk - a utility for remote partitioning of hard disks on the compute nodes
     beoserv - the beoboot server; it responds to compute nodes and serves the boot image
     bpmaster - the bproc master daemon; it only runs on the master node
     bpslave - the bproc compute daemon; it runs on each of the compute nodes
     bpstat - a bproc client; it maintains status information for all nodes in the cluster
     bpctl - a bproc client; a command line mechanism for controlling the nodes
     bpsh - a bproc client; a replacement utility for “rsh” (remote shell)
     bpcp - a bproc client; a mechanism for copying files between nodes
     beompi - an optimized implementation of the Message Passing Interface
     beopvm - the Parallel Virtual Machine; optimized for use with Scyld Beowulf
     mpprun - a parallel job creation package for Scyld Beowulf
     perf - support for platform specific hardware performance counters

    فعال سازی نرم افزار درصدگیری از کیفیت و کمیت اطلاعات در وب سایتها و صفحات شخصی اساتید

    مرکز انفورماتیک نرم افزاری تحت وب جهت درصدگیری اتوماتیک کیفی و کمی وب سایت های بخشهای مختلف را طراحی و آماده نموده است که در سال جاری فعال خواهد شد و گزارشات حاصله از این نرم افزار، جهت تعیین میزان فعالیت بخشهای مختلف و همچنین تشویق وب سایتهای برتر، در وب سایت اطلاع رسانی داخلی دانشگاه (http://info.znu.ac.ir) منعکس خواهد شد.
    لازم به ذکر است، روال درصدگیری این نرم افزار در دو مرحله به شرح ذیل صورت خواهد گرفت:

     درصدگیری کمی: دسترسی های لازم به رابطین وب داده خواهد شد تا همزمان با بروزرسانی صفحات، تاریخ و اسامی ارائه کنندگان اطلاعات را در نرم افزار مذکور وارد نموده تا درصد گیری از کمیت و میزان بروز بودن اطلاعات توسط بخشها بصورت اتوماتیک انجام شده و قابل نمایش در سایت داخلی دانشگاه بصورت گزارشهای هفتگی، ماهانه و سه ماهه باشد.

     درصدگیری کیفی: در این نرم افزار امکان نظردهی هر یک از رابطین در رابطه با کیفیت عملکرد سایر بخشها در زمینه وب سایت، در نظر گرفته شده است، به گونه ای که هر ماه رابطین با توجه به آیتمهای مربوط به کیفیت که در نرم افزار لحاظ شده است، نظر خود را نسبت به هر مورد در قالب درصد درج نموده و نرم افزار با برآورد کلیه نظرات، درصد نهایی را نمایش می دهد.

    ضمناً امکانات مذکور برای سیستم اساتید نیز در نظر گرفته خواهد شد.
